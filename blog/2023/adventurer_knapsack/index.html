Adventurer's Knapsack

June 3, 2023

2023   ·   crypto  

Background

For every good trip in the afterworld you need a good knapsack!

from SECRET import flag, privk, m, n
from sage.numerical.knapsack import Superincreasing

flag_int=bytes_to_long(flag)
L=len(flag)*8
assert L==176

assert Superincreasing(privk).is_superincreasing()
assert m > sum(privk)
assert gcd(n,m) == 1

pubk= [(n*i)%m for i in privk]
shuffle(pubk)

ct=0
for i in range(L):
    if flag_int & 2^(L-i-1) != 0:
        ct += pubk[i]

print(f'{ct=}')
print(f'{pubk=}')

privk 수열이 superincreasing을 만족한다.

Superincreasing 이란

양의 실수 \(S_1, S_2, S_3 …\) 에 대해서 \(S_{n+1} > \sum_{j=1}^n S_j\) 를 만족하는 수열을 말한다.

예를 들어 (1, 3, 6, 13, 27, 52) 가 있다.

sage에서 다음과 같이 사용한다.

privk 리스트가 superincreasing을 만족한다.

n과 m은 서로소이며, m은 sum(privk) 보다 크다.

pubk는 각 privk 요소에 n을 곱하고 m으로 나눈 값이다. n이 뭔지는 모른다.

flag_int의 상위 비트부터 i로 보면서 해당 값이 1이면 ct에 pubk[i]를 더한다.

찾아보니 Merkle–Hellman knapsack cryptosystem 인 것 같다.

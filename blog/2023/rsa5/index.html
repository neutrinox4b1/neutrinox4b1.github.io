같은 n에 대해 서로 다른 공개 키 e1, e2로 암호화된 암호문 c1, c2를 제공한다.

평문 \(m\) 에 대해

\[c_1 = m^{e_1} \bmod n\] \[c_2 = m^{e_2} \bmod n\]

\(e_1, e_2\) 는 서로소이다. \(\therefore \gcd(e_1, e_2) = 1\)

따라서 EEA를 이용해서 베주항등식 \(e_1s + e_2t = 1\) 을 만족하는 s, t를 찾을 수 있다.

그렇게 되면 \(c_1^s c_2^t = m^1\) 이 되어 평문을 구할 수 있다.

def EEA(r1, r2, u1, u2, v1, v2) :
    if r2 == 0:
        print(f'gcd: {r1}\nu: {u1}\nv: {v1}')
        return
    q = r1//r2
    r = r1%r2
    u = u1 - q*u2
    v = v1 - q*v2

    return EEA(r2, r, u2, u, v2, v)

#EEA(e1, e2, 1, 0, 0, 1)

print(long_to_bytes(m))
